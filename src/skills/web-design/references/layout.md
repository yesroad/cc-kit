# 📐 레이아웃 & 비주얼 스타일 트렌드 (2025–2026)

## 📐 레이아웃 트렌드

### 1. 벤토 박스 레이아웃 (Bento Box Layout)

```
┌─────────────────┬────────┬────────┐
│                 │        │        │
│   Hero Card     │  카드   │  카드  │
│    (2×2)        │  (1×1) │  (1×1) │
│  가장 중요한       ├────────┴────────┤
│    콘텐츠         │   Wide Card     │
├────────┬────────┤    (2×1)        │
│  카드   │  카드   │                 │
│  (1×1) │  (1×1) │                 │
└────────┴────────┴─────────────────┘
```

다양한 크기의 카드를 격자에 빈틈 없이 배치. 중요한 항목은 크게, 보조 항목은 작게 → 레이아웃 자체가 정보 위계를 표현

**쓰이는 곳:** Apple 제품 소개, Linear, Vercel 랜딩페이지, iOS 위젯 화면

### 2. 탈구축(Deconstructed) 히어로 레이아웃

```
┌──────────────────────────────────┐
│  BRAND   ← 작은 레이블           │
│                                  │
│        ╔══════╗                  │
│  BIG   ║IMAGE ║  OVERLAPPING     │
│  TYPE  ║      ║  TEXT            │
│        ╚══════╝  ← 비대칭        │
│           DATA VIZ ↗             │
│           ← 기울어진 요소들      │
└──────────────────────────────────┘
```

대칭·정렬을 의도적으로 깨서 실험적·독창적 인상을 줌. 텍스트·이미지·데이터 시각화가 겹치고 기울어지며 배치됨

---

### 3. Above-the-Fold 집중 레이아웃

```
┌──────────────────────────────────┐  ← 스크롤 없이 보이는 영역
│                                  │
│   핵심 헤드라인 (크고 굵게)      │
│   서브 메시지 (간결하게)         │
│                                  │
│   [ CTA 버튼 ]  ← 여기 있어야 함│
│                                  │
└──────────────────────────────────┘  ← fold line
  │  스크롤 후 보이는 추가 정보... │
```

CTA가 fold 위에 있을 때 전환율 **304% 우위**. 핵심 정보만 위로 올리는 설계

## 🎨 비주얼 스타일 트렌드

### 4. Bold + Kinetic Typography

```
┌──────────────────────────────────┐
│                                  │
│         MASSIVE                  │
│         HEADLINE  ← 뷰포트를 꽉 채움
│         TEXT                     │
│                                  │
│  ↕ 스크롤하면 텍스트가 움직임    │
│    → 슬라이드 인                 │
│    ← 페이드 인                   │
│    ↻ 회전                        │
│                                  │
└──────────────────────────────────┘
```

타이포그래피 자체가 비주얼. 이미지 없이도 임팩트 있는 첫인상 가능

**혼합 폰트 예시:**

```
BOLD DISPLAY FONT   ← 헤드라인 (굵고 크게)
Regular Serif       ← 본문 (읽기 쉽게)
```

### 5. 뉴모피즘 2.0 (Neumorphism 2.0)

```
기존 플랫 디자인:           뉴모피즘 2.0:
┌──────────────┐            ╔══════════════╗
│    버튼      │  →         ║              ║  ← 위쪽: 밝은 그림자
│  (그냥 평평) │            ║    버튼      ║  ← 배경과 같은 계열 색
└──────────────┘            ║              ║  ← 아래쪽: 어두운 그림자
                            ╚══════════════╝
                              마치 배경에서 돌출된 느낌
```

기존 뉴모피즘의 접근성 문제(낮은 대비) 개선 + 미니멀리즘 결합. 실물감 있되 과하지 않음

---

### 6. 클레이모피즘 (Claymorphism)

```
      ╭──────────╮
   ╭╯  😊  3D  ╰╮  ← 둥글둥글한 3D 점토 느낌
   │   캐릭터    │  ← 부드러운 그림자
   │   오브젝트  │  ← 채도 높은 색상
   ╰──────────╯
   ↑
   inflate된 느낌, 눌릴 것 같은 질감
```

앱 온보딩, 빈 상태(empty state) 화면, 게임형 UI에서 친근하고 유쾌한 분위기 연출

---

### 7. 미니멀 브루탈리즘 (Minimalist Brutalism)

```
┌──────────────────────────────────┐
│                                  │
│   BIG BLACK TEXT                 │
│                                  │
│   ━━━━━━━━━━━━━━  ← 굵은 선      │
│                                  │
│   [ BUTTON ]  ← 테두리만 있는 버튼│
│                                  │
│   #000000 / #FFFFFF / #FF2800만 씀│
│                                  │
└──────────────────────────────────┘
↑ 장식 없음. 기능만 남김
```

꾸밈을 철저히 제거. 날것의 기능성 강조. 데이터 시각화·대시보드에서 특히 효과적

### 8. 레트로 리바이벌 (Retro Revival)

| 시대 | 가져오는 요소              |
| ---- | -------------------------- |
| 70s  | 어스톤 컬러 + 그루비 폰트  |
| 80s  | 네온 + 그리드 라인         |
| 90s  | 픽셀 아트 + 윈도우 UI 느낌 |
| 00s  | Y2K 메탈릭 + 버블 버튼     |

↓ 현대적으로 재해석 → **깔끔한 레이아웃 + 레트로 요소 포인트**

불안한 현실에서 친숙하고 안정적인 것에서 위안을 찾는 심리 반영
